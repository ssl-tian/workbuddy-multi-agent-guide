## 内容结构

### 诊断+病因+流行病学模块（Day 1/3/5/7/9）

```markdown
# [病种名称] — 诊断·病因·流行病学

## 📋 今日导读
> 一句话概括，阅读时长预估。

## 1. 诊断标准（占40%篇幅）
### 1.1 ICD-10 诊断标准
逐条列出核心症状、病程标准、排除标准、严重程度标准。

### 1.2 🆕 ICD-11 变更要点
> 显著标记格式。3-5个关键差异及临床影响。

### 1.3 鉴别诊断
表格形式，至少3个需鉴别疾病，附核心鉴别要点。

## 2. 病因假说（占25%篇幅）
依次覆盖：生物因素（神经递质/神经内分泌/神经影像）、遗传因素、心理社会因素、最新病因模型。每个维度2-3句。

## 3. 流行病学（占25%篇幅）
终生患病率（中国数据优先）、性别差异、发病年龄分布、地域/文化差异、共病率。

## 4. 📝 今日要点速记
3-5条核心知识点总结。

## 5. 📖 推荐阅读
指南出处、教科书参考。
```

## 内容质量标准

1. **权威性**：诊断严格引用ICD-10措辞，药物方案引用中国指南推荐
2. **准确性**：药物通用名+商品名、剂量、用法与最新指南一致
3. **可读性**：表格化关键信息，5000-7500字，20-30分钟阅读量
4. **实用性**：标注"规培考试高频考点""易混淆点"

---

## 知识来源优先级

1. 精神障碍诊疗规范（2020年版）— 国家卫健委
2. 中国各病种防治指南（第二版/2025版）— 中华医学会精神病学分会
3. 郝伟主编《精神病学》（第8版）— 规培指定教材
4. ICD-10 / ICD-11 — WHO
5. 国际指南（CANMAT / NICE / APA）— 补充参考

---

## ⛔ 可信度铁律

- 所有药物方案必须来自上述权威来源，绝不编造
- 无法确认的信息标注「待查证」
- 诊断标准不可简化到失准
- Day 11 新知速递：所有 DOI 必须逐条交叉验证（CrossRef / 原文页面）

---

## 进阶规则（v4 第 2 周期新增）

- 2025 版中国指南优先于旧版（精分/抑郁 2025 版覆盖第二版）
- ICD-11 CDDR 2024 年 3 月版优先于 ICD-11 初版
- GBD 2021 数据优先于 GBD 2019
- ENIGMA mega-analysis 优先于单中心影像学研究
- 新增假说仅在原文有 >1 项独立实验室证据时纳入
- 输出前执行 5 项自查：DOI真实性 / 方向性错误 / 来源一致性 / 字数 / 覆盖度
- 文件命名：`精神医学规培_Day{N}_{病种}_{日期}.md`
- 标记要求：🆕 = 本次新增、⚠️ = 易混淆点、🚨 = 安全警告

---

## 输出操作

1. 生成完整 Markdown 内容（严格按模块模板）
2. 文件命名：`精神医学规培_Day{编号}_{病种}_{日期}.md`
3. 报告今日生成的病种和模块名称
4. 上传至 IMA 知识库「B there 的知识库」→「精神医学规培」文件夹
5. 如 IMA API 认证失败，仅保存本地，标记等待手动上传
