# 精神医学规培每日自动化

> 真实运行的 WorkBuddy 自动化任务 — 每天自动生成精神科知识笔记并上传 IMA 知识库

## 是什么

一个 11 天轮换的每日自动化任务，覆盖精神科 5 大病种（精分/抑郁/双相/酒精/焦虑），每天生成 ~7,500 字知识笔记并自动上传到手机端知识库。**已稳定运行 20+ 天，产出 20 篇笔记。**

## 快速复用

1. 打开 [WorkBuddy](https://www.codebuddy.cn)
2. 复制 [`automation-prompt.md`](./automation-prompt.md) 中的完整 prompt
3. 粘贴到 WorkBuddy 对话框，修改病种/日期/输出路径为你的领域
4. 设置定时：`automation_update` → 每日 16:00 → `激活`
